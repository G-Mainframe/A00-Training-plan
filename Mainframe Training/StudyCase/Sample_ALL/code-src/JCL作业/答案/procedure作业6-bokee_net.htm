字体大小：大 | 中 | 小 2007-01-14 19:59 - 阅读：36 - 评论：0

6． 新建自己的PROCLIB，把练习5中的procedure变成cataloged procedure，然后修改练习5的作业，改为调用cataloged procedure，将输出数据集改为userid.JCLW3D1.PS61和userid.JCLW3D1.PS62，看看JOBLOG有什么不同。比较练习5和练习6的作业，体会两种procedure的特点。

注：USER PROCLIB调用举例

在WISCO15.EXERCISE（PROC1）中输入PROC1：

000001 //PROC1 PROC UN=3390,U=TRK,                   

000002 // SR=TSTWK1,S=10,P=20                        

000003 //STEP1 EXEC PGM=IEBGENER                     

000004 //SYSPRINT DD SYSOUT=*                        

000005 //SYSUT2   DD DSN=&D,UNIT=&UN,                

000006 //            VOL=SER=&SR,SPACE=(&U,(&S,&P)), 

000007 //            DISP=(,CATLG),                  

000008 //            DCB=(RECFM=FB,LRECL=80,DSORG=PS)

000009 //SYSIN    DD DUMMY                           

000010 // PEND                                        

在WISCO15.EXERCISE（PROC2）中输入PROC2：

000001 //PROC2 PROC                         

000002 //STEP EXEC PGM=IEBGENER             

000003 //SYSPRINT DD SYSOUT=*               

000004 //SYSUT1   DD DSN=&D,DISP=OLD        

000005 //SYSUT2   DD SYSOUT=*               

000006 //SYSIN    DD DUMMY                  

000007 // PEND                              

在WISCO15.EXERCISE（CATALOG）中输入作业六：

000001 //SYMBOL JOB (),'DORIS',                               

000002 //         MSGCLASS=X,                                  

000003 //         CLASS=A,                                    

000004 //         MSGLEVEL=(1,1),                             

000005 //         NOTIFY=WISCO15                              

000006 //MYLIB  JCLLIB ORDER=WISCO15.EXERCISE                 

000007 //S1 EXEC PROC1,D='WISCO15.JCLW3D1.PS61'               

000008 //SYSUT1 DD *                                          

000009 HELLO                                                  

000010 //STEP2 EXEC PROC2,D='WISCO15.JCLW3D1.PS61'            

000011 //STEP3 EXEC PROC1,D='WISCO15.JCLW3D1.PS62',           

000012 // SR=TSTWK2,S=2,P=4                                   

000013 //SYSUT1 DD *                                          

000014 HELLO                                                   

000015 //STEP4 EXEC PROC2,D='WISCO15.JCLW3D1.PS62'
