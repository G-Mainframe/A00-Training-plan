本書は、以下に示すシステムのもとで動作するソフトウェアComponetAA ClientX 運用パッケージ V1.0L10に含まれる帳票印刷コントロールの説明書です。

  • Microsoft(R) Windows(R) 98 operating system

  • Microsoft(R) Windows NT(R) Workstation operating system Version 4.0
  • Microsoft(R) Windows NT(R) Server Network operating system Version 4.0
  • Microsoft(R) Windows NT(R) Server Enterprise Edition4.0
  • Microsoft(R) Windows NT(R) Server 4.0 Terminal Server Edition
  • Microsoft(R) Windows NT(R) Server 4.0 Terminal Server Edition SP4以上 + Citrix MetaFrame 1.8J
  • Microsoft(R) Windows(R) 2000 Professional operating system
  • Microsoft(R) Windows(R) 2000 Server operating system

本書は、帳票印刷コントロール機能の使用方法について説明しており、帳票定義体を使って出力プログラムを作成する方を対象としています。

■本書の目的

本書では、帳票印刷コントロールについて説明します。

帳票印刷コントロールが提供するプロパティ、メソッド、イベントに関しては、帳票印刷コントロールリファレンスを参照してください。

帳票定義体とオーバレイ定義体の作成方法については、“PowerFORMヘルプ”を参照してください。帳票を印刷するために必要な MeFtについては“MeFt 説明書”を参照してください。
