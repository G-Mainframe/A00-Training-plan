■SQL Serverデータについて
 次のファイルは、第5章、第7章、付録Dで利用するデータベースを構築するためのスクリプトです。
 ・cobdeb.sql
 ・setupcobdb.bat
 setupcobdb.batをコマンドラインで実行することにより、SQL Server 2005 Expressにデータベースが作成されます。

■XLSファイルについて
 次のファイルは、第5章、第7章、付録Dで利用するデータベースに格納されているデータのサンプルです。使用するデータベースがSQL Server以外の場合、データベース作成のサンプルとして利用してください。
 ・SYAPF.xls：社員テーブル用データ
 ・SYOPF.xls：商品テーブル用データ
 ・SYOIOPF.xls：入出庫テーブル用データ
